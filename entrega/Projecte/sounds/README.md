Drop your runtime audio files here.

Expected filenames (optional):
- music_bg.mp3
- key_pickup.wav
- sword_pickup.wav
- heal_pickup.wav
- shield_pickup.wav
- teleport.wav
- jump.wav
- hurt.wav
- sword_attack.wav

The game tries paplay, aplay, ffplay, and play (SoX) in that order.
If a file is missing, that sound is silently skipped.
